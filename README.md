# DataQualiityCheck

## 요구사항
- python >= 3.7.1
- git 

## 설치

### venv 설치 및 활성화 
```
virtualenv pre_process 

cd pre_process 
source bin/activate
```

### git clone 
```
git clone https://github.com/Leejung8763/DataQualityCheck.git
```

### pip install 
```
pip install -r requirements.txt 
```

### folder setting
```
data/
├── documents/
└── data.csv
```   
    
## 예제 실행 
```
python sampleCode.py 
```
